STM32CubeG0 Firmware Package V1.4.0 / 03-November-2020

Main Changes
  1. Add support of STM32G0C1xx/B1xx/B0xx devices (CMSIS/HAL drivers)
  2. Add support of STM32G061xx/51xx/50xx devices (CMSIS/HAL drivers)
  3. Add new middlewares: Openbootloader, USB devices & USB host
  4. [FreeRTOS] Support of new R0.12c (ST modified 20191011)
  5. [FatFs] Support of new v10.3.1_20200831
  6. [USBPD] Support of new USBPD core library (v3.2.0) 
  7. [USBPD] Support of new USBPD devices library (v3.1.0)
  8. [Utilities] Update to latest GUI_INTERFACE & TRACER_EMB version
  9. Add New BSP driver for STM32G0C1E-EV board
  10. Add new FW examples/application/demo for STM32G0C1E-EV & NUCLEO-G0B1RE boards
  11. Add support of STM32CubeIDE for all devices/projects
